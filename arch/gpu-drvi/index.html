archLinux 下显卡驱动相关整理 intel amd Nvidia 包括虚拟机的

2022-12-19
阅读时长: 1 分钟

mesa

sudo pacman -S mesa 它提供用于3D加速的基础DRI驱动程序。

虚拟机下的显卡驱动

虚拟机的情况，有很多中，常用的 vmware 和 virio gpu

virio gpu

需要手动 编译mesa

vmware / virtual box

sudo pacman -S xf86-video-vmware

intel核显

sudo pacman -S xf86-video-intel

Ivy Bridge及以上架构的cpu 只安装vulkan-intel 也可以

Nvidia

sudo pacman -S nvidia nvidia-prime nvidia-settings nvidia-utils opencl-nvidia lib32-nvidia-utils lib32-opencl-nvidia

nvidia驱动非开源的 可能需要/etc/pacman.conf打开 multilib 仓库 另外可能需要

sudo pacman -S xf86-video-nouveau
