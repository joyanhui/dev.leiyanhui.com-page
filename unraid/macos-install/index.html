unraid 安装macos的注意事项

2022-10-29
阅读时长: 1 分钟

基本安装

用Macinabox 插件就可以，需要提前安装 user script插件，然后 等容器执行完成后，再去 插件>user script 执行一次 1_macinabox_helper

就自动创建虚拟机了。

常见问题

编辑虚拟机配置后 无法启动

因为unraid的图形模式编辑macos的kvm配置文件，是会丢失一些配置的。所以 所有修改，都要切换到xml视图手动编辑。

我这里有一个默认的xml的配置 点这里查看

无限重启，开机五国代码错误

多是 macinabox 创建的opencore太旧了，macos有更新。下载最新的替换掉就好。

https://github.com/thenickdude/KVM-Opencore/releases

注意： 下载回来的文件是iso后缀，但是其实是raw格式的img文件，自己重命名一下

macos13.0 的问题

你可以 创建一个docker 跑一个linux系统，然后直接获取 macos13的恢复镜像，并使用最新的 kvm-opencore 引导

最后，经过我测试，pve下安装好的macos 其实可以直接copy到unraid之类启动的
