尝试记录

环境 arch 2022-10-18

先尝试  安装一个图形界面的远程管理工具吧。我这里选择 Remmina

1

sudo pacman -S  remmina

默认 只支持 ssh x2go

如果你需要 RDP 支持，还要选择安装 freerdp 或 remmina-plugin-rdesktop(AUR)。

1
2
3

yay -S remmina-plugin-rdesktop
# 关闭remmina
killall remmina

测试后 发现好像连不上

1
2

yay -Rs remmina-plugin-rdesktop
sudo pacman -Rs rdesktop

试试 freerdp

1
2

sudo  pacman -S freerdp
killall

再次使用 可以正常链接Windows了

rdp文件也可以自动关联到remmina

但是 remotapp 依旧不知道应该如何支持
