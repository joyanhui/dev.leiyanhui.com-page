dwm 扩展 安装方法

首先用 卸载yay安装的版本，用编译方式安装的dwm

创建目录 用于存补丁

1
2
3

mkdir  ~/dwmfiles/patches/
#没有安装wget也安装一下
sudo pacman -S wget

获取补丁

官网地址 https://dwm.suckless.org/patches/

类似一个应用市场

找到 https://dwm.suckless.org/patches/systray/

我现在的是最新版本 6.3版 所以就下载他对应的补丁

1

wget https://dwm.suckless.org/patches/systray/dwm-systray-6.3.diff

到dwm源码目录

1

mkdir  ~/dwmfiles/dwm/

如果么有配置git 用户的话配置一下，

1
2

 git config --global user.name "leiyanhui"
 git config --global user.email "leiyanhui@xxx.com"

打补丁，重新提交commit 编译

1
2
3
4

patch < ~/dwmfiles/patches/dwm-systray-6.3.diff
git commit -am "systray"
rm -rf config.h # 这个文件是每次编译的使用 那个 config.ref.h自动生成的
sudo make clean install  #重新安装
