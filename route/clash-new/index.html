clash删库事件发生后的替代方案

2023-11-08
阅读时长: 1 分钟

因为 clash for windows 作者自我开盒（不小心泄露车牌号码）随后删库导致的连锁反应，clash和clash meta 等相继把仓库设置为归档并暂时停止更新。 其实clash for windows 只是clash一个在windows的ui配置工具 ui部分基于Electron,后端还是 clash和clash meta。但是因为他比较直观并且很美观，所以使用者众多。

目前暂时看 clash和clash meta ，应该不会一直停止更新。所以可以继续使用。

重点：clash 和clash-mete clash-tun 依旧可以正常使用！！！

clash meta的优点 有很多，主要分流规则完善且灵活，并且支持的协议多，基本上覆盖所有协议。但其实有两个问题。 1、在低配硬件上跑不满带宽，甚至只有带宽的10-20%
2、配置较为繁琐，对新手不友好，甚至不少大神也对clash的配置很懵

主流的替代软件客户端内核 比如 v2ray xray sing-box Hysteria,只是其他软件的分流和协议支持度或者易用性都没有clash好。
这些可以不操心，其他非基于clash的客户端有：

V2rayN (Win端)https://github.com/2dust/v2rayN/wiki/List-of-supported-cores
v2rayA( linux + win + macos)
v2flyNG(android) V2rayNG (android)
Nekoray (win+linux) https://github.com/MatsuriDayo/nekoray
Nekobox (android)
Surge (收费 macos ios) sing-box（全平台） IOS端：sing-box Shadowrocket QuantumultX Surge等

上面支持linux的在openwrt上都可以用，但是另外有一些带luci界面： PassWall Bypass SSR-Plus PassWall2 VSSR (Hello World)，

最后： 较为看好sing-box,已经有部分机场开始支持（也有一些clash转换sing-box的转换工具）ios客户端是免费的。
