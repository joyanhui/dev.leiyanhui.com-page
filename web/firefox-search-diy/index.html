firefox自定义搜索引擎

2023-11-10
阅读时长: 1 分钟

firefox的新版本目前已经不支持自定义任意搜索引擎，即 自定义 https://url?keyword={关键词} 这种格式。 而 chrome和edge都可以灵活自定义。

firefox 旧版本是支持的，但是知道什么时候 更新后就不支持了。只能用插件方式安装，但是插件是支持的搜索引擎就那么几个也不是我们常用需要自定义的。

找到一个曲线解决方法。 利用firefox的地址栏可以搜索书签 以及 书签可以定义关键词的功能。

以bing 换到cn.bing.com为例，firefox默认是调用 www.bing.com 在某些环境中很卡。

网址如下： https://cn.bing.com/search?q=111

把这个地址添加到书签，名称随意，地址修改为 https://cn.bing.com/search?q=%s 关键词输入 cnbing（和其他的不冲突即可）

然后地址栏输入 cnbing+空格+关键词 即可调用cn.bing.com 搜索对应关键词

同样可以添加 github 个人博客 等等 所有支持get方式的搜索

这个方案的缺点是 没法省略前面的关键词和空格 作为默认搜索引擎使用。

吐槽： firefox 真的能砍，砍掉了渐变app（webapp）又砍自定义搜索引擎，如果不是因为他在linux下支持硬解，真的够够的的。

edge的linux版本的垂直标签功能已经非常完善，不经常在线看视频的话，早点离开吧。

附：
跨浏览器 跨设备同步书签和密码插件推荐：

  • floccus（需要一个webdav同步 可以用nas或者坚果云）
  • bitwarden （可以自建服务端，全平台可用 ios端为数不多的体验不错的密码管理器）
